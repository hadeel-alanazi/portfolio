Project Information

  • Category: FinTech Web Application
  • Project Date: 2025 (AMD Hackathon)
  • GitHub Repository: github.com/hadeel-alanazi/maly

Summary

Maly is a smart financial assistant web application designed to help users manage their personal finances with ease. The platform allows users to track expenses, monitor income, set financial goals, and receive AI-powered recommendations based on their spending patterns.

Features

  • User registration and secure authentication
  • Interactive financial dashboard
  • Add and categorize expenses
  • Set and track financial goals
  • AI-powered financial recommendations using OpenAI GPT
  • Responsive and user-friendly interface

Technologies Used

  • PHP
  • MySQL
  • Bootstrap 5
  • Chart.js
  • OpenAI API
